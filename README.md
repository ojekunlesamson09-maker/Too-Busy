# Too Busy.
### AI replies for people who can't be bothered

Paste a message. Pick your vibe. Copy a nonchalant reply and get on with your life.

## Vibes
- **Softly Busy** — warm but gently unavailable
- **Lowkey Unavailable** — chill and unbothered
- **Chronically Booked** — politely overwhelmed

## Deploy

1. Clone this repo
2. Install Vercel CLI: `npm i -g vercel`
3. Add your `GROQ_API_KEY` as an environment variable in Vercel
4. Run `vercel` to deploy

## Built with
- Vanilla HTML/CSS/JS
- Groq API (llama-3.3-70b)
- Vercel serverless functions
